# Wild-Tiger Restaurant and Bar

## Features
 - 100% mobile responsive
 - Admin item creator
 - Dynamic menu items from a sql lite database and an option to fallback to a saved JSON
 - Responsive UI and UX
 - Smooth transitions and animations
